# HANDOFF — состояние проекта и план (обновлено после сессии 2026-07-04)

Файл для передачи контекста между сессиями AI-агентов. Прочитай его вместе с

## Что это за проект

«Дальняя Марка» — браузерная simulation-игра одним HTML-файлом (владелец
тестирует с телефона Android, открывая `index.html`). Поселение переселенцев
на острове: автономная экономика (job pool + рынок труда), автостроитель,
герои/экспедиции/бои, дань империи, рейды. Код: модули `src/*.js` в общем
scope (НЕ ES-модули), склейка `node build.mjs` → `index.html` + `dist/game.js`.

## Сделано в этой сессии (коммиты 5741cc2..faa34a7)

**Инфраструктура**
- Монолит распилен на ~30 модулей `src/` + сборщик `build.mjs`; headless-тест
  `node dist/game.js <seed> [quest]`; `docs/function-index.md` автогенерится.
- Перф: `paintFog` 550мс → ~10мс (LUT пиксель→гекс + ImageData + кромочное
  смягчение); террейн-слои статичны кроме леса, список треугольников кэширован.

**Геймплей (11 фич первого пакета)**
- Дороги без «паутины»: тянутся к ближайшей подключённой дороге; у гекса
  строения максимум одно ответвление-«подъезд» (визуально).
- Лачуга (2 жителя) апгрейдится в дом тир-2 (3 жителя, спрайт `b_house2`).
- Таверна варит эль из зерна ферм (`b.ale`, `CFG.STORE.tavern`); жители пьют
  эль ночью; без действующей фермы эль кончается.
- Порт: корабль — отдельная стройка (`CFG.SHIP`, 28 дерева, верфь, трюм 24);
  морская торговля только рейсами; ИМПОРТ работает: заказ → рабочий грузит
  золото из казны в `b.holdGold` → рейс привозит товар. Караваны гильдии —
  мгновенная торговля без порта.
- Шахты: конечная руда (`b.data.oreLeft`), самоцветы только с тира 2;
  выработанная шахта = заброшена навсегда = делв-энкаунтер для героев
  (в работающую спуск запрещён). Бедные «холмовые» шахты на скалах (ROCK).
- Навыки жителей (`src/10b_skills.js`): 7 навыков качаются работой, влияют
  в основном на бой героев (Секира/Крепь/Инженерия/Жилы/Знахарство/Дозор/
  Тайноведение); топ-3 в инспекторе.
- Экран боя (`16b_battle.js` ядро + `22b_battle_ui.js` UI): Disciples-стиль,
  передний/задний ряд, заклинатели бьют любой ряд, саппорт лечит; враги:
  гоблин/шаман/разбойник/скелет/некромант/зверь/огненный и магмовый атронахи;
  составы по типу логова и этапу; headless крутит то же ядро мгновенно;
  вызовы через `beginBattle(opts, continuation)`.
- Новое логово «Звериное логово» (den) в лесу.

**Реки v2 (по границам гексов, референс Оскара Столберга)**
- Река идёт по dual-сетке: ребро→центр треугольника→ребро. Данные:
  `S.riverEdges` (Set заблокированных РЁБЕР между парами гексов),
  `S.riverTris` (тайлы `rt_{терр}_{ориент}_{маска}`, берега под террейн),
  оверлеи `r_falls` (исток-водопад) и `r_mouth` (устье с пеной).
- Проходимость: переход через речное ребро закрыт; МОСТ = дорога на ОБОИХ
  берегах ребра (проверка в `findPath`). Автостроитель планирует мосты
  (`withBridgedPass`, план с `bridge:true`, фундамент ждёт через `waitBridge`)
  и как срезку большого крюка (обход длиннее моста в 2 раза).
- Дерево рек: entity `S.world.rivers` c parent/child; точка слияния принимает
  ровно одну реку; самослияние запрещено; тупик → пруд; сток озера ≤ 1.

**Генератор мира (`src/02c_worldgen.js`)**
- Пайплайн: остров (море со всех сторон, один массив суши, `S.distCoast`) →
  озёра среднего пояса (2-4) → хребты-«снежинки» (2-3 тонкие цепи с отрогами;
  инвариант: пик соседствует только с хвостом; узлы по степени; entity
  `S.world.ranges`) → высота (`S.elev`: берег→хребет) → реки (5-7, вниз по
  высоте) → WFC-заполнение GRASS/FOREST/ROCK: волна ОТ КОНТУРОВ внутрь,
  вес = зональный приор (прибрежные луга/утёсы, лесной пояс, поймы,
  предгорья) × аффинность схлопнутых соседей; правила мягкие, GRASS —
  запасной тайл (противоречий нет по построению) → релаксация (цельные
  леса, без одиноких скал) → entity лесов/озёр → валидация с retry.
- Дебаг: кнопка «WFC-реплей» в панели отладки (~) — пошаговый проигрыш
  волны, красные вспышки = релаксация, в конце рёбра рек. Флаг `WFC_DEBUG`
  (вне S), трейс в `S.wfcTrace`.

**Крупные правила мира**
- Масштаб: карта 64×64, гексы визуально ×2 (`ZOOMS [4,6,8]`), юниты рисуются
  в полгекса (экранный размер прежний). Спрайты растянуты ВРЕМЕННО — в плане
  перерисовка под новый масштаб.
- Лес непроходим для ЖИТЕЛЕЙ (`S.pass`), герои/партии/рейдеры ходят
  (`S.passHero`, `withHeroPass`). Пеньки проходимы; лес регенерирует из
  пеньков (15-23 дня); лесоруб рубит с опушки (adj).
- Лесопилки: прогрессивная стоимость (`costOf`): 0 действующих → бесплатно
  (фикс дедлока дерева); заброшенные не оживают (энкаунтеры); фермы/причалы
  оживают при восстановлении угодий.
- Старт: палатка переселенцев → фаза `S.phase='scout'` (разведка радиус 6,
  экономика спит) → игрок ставит ратушу тапом (валидация `thSiteScore`,
  кнопка «Авто»; headless — авто через 2 дня) → палатка сворачивается →
  `'play'`. «Губернаторские указы» ensureStarter* удалены.

## Известные хвосты / не отполировано

- Спрайты растянуты ×2 — ждут перерисовки под новый масштаб (владелец в курсе).
- Баланс нового мира не вылизан: плотность лесов/скал, темп экономики при
  непроходимом лесе, дальность шахт у гор — крутить по фидбеку с телефона.
- Тап в фазе выбора ратуши ставит её сразу (без подтверждения) — возможны
  мисклики на телефоне; кандидат: двухтаповое подтверждение.
- В бэклоге WFC: жёсткие правила соседства (появится настоящий бэктрекинг —
  каркас трейса уже пишется), больше типов тайлов.
- `19_atlas.js`/`20_render.js` крупные — пилить по подсекциям при росте.
