КРОНОВАЯ МАСКА

клубы / псевдошары разного размера · взвешенный Worley + smooth-max · бесшовно тайлится
Вершины крон разбросаны по ячейкам (хэш в мировых коорд. → маска тайлится). В каждой — радиальный купол своего радиуса; поле = плавный максимум куполов: при слитности 0 — отдельные кроны с границами-водоразделами (как watershed из LiDAR), при росте — сливаются в клубы. Порог задаёт покрытие (лес↔поляны).
—
купол i: Rᵢ−|p−vᵢ| · поле = smax(куполов, k) · граница = взвешенная диаграмма (Аполлония) · хэш вершин по mod-ячейкам ⇒ бесшовный тайл
