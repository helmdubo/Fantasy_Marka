# Дальняя Марка (Fantasy Marka)

Прототип simulation-игры в браузере: пограничная марка на краю империи.
Поселенцы разных рас сами строят, добывают и воюют; игрок наблюдает и направляет.
Давление внешнего мира — голод, рейды логовищ, имперская дань. С гильдией
авантюристов появляются герои: разведка, штурмы, обнос логовищ, спуск в шахты.

## Запуск

Открыть **`index.html`** в браузере — это один самодостаточный файл (нужен
интернет для three.js с CDN). Работает и на телефоне.

## Разработка

Код разбит на модули в `src/`, из которых собирается `index.html`:

```
node build.mjs                 # пересобрать index.html после правок src/
node dist/game.js test-1       # headless smoke-test симуляции (без браузера)
node dist/game.js test-1 quest # + сценарий героев и логовищ
```

`index.html` не редактируется руками — только генерируется. Подробности
устройства кода — в [ARCHITECTURE.md](ARCHITECTURE.md), индекс функций —
в [docs/function-index.md](docs/function-index.md).
